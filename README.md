# Bias-Detection-System-for-Pakistani-News
Bias Detection System for Pakistani News
1. Download edge-driver and specifiy its path in edge_driver_setup.py.
2. Run main.py in data_scraping folder to scrape data for dataset
3 .csv files will be created, move those files to data_initial_processing folder.
3. Run data_process.py in inital_processing folder a .csv file will be created.
4. Move that newly created .csv file to data_preprocessing folder
5. Run text_vectorization.py, data_processing.py and visualizations sequentially.
